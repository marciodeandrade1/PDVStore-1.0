# PDVStore

Principais Classes por Pasta (Resumo Completo)

Models
	Entidades do banco de dados, como Produto, Venda, Cliente, etc.

Data
	DbContext, Migrations, Seeders, etc.

Services
	Finalizar venda, atualizar estoque, calcular total, etc.
	Repositorios e gráficos, Mock PIX/Cartão/Dinheiro

ViewModels
	Dados que serão exibidos nas Views, como listas de produtos, detalhes de vendas, graficos, etc para o Dashboard.

Forms
	Telas de cadastro, edição e visualização de dados.

Helpers
	Centralizar connection string, funções auxiliares, validações, formatações, etc.
